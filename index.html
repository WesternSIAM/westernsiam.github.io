SIAM (Society for Industrial and Applied Mathematics)

This will be a way for students from various departments that enjoy mathematics and computation as well as interdisciplinary research, to get together and learn more about things like scientific computing for example using Python, Matlab, machine learning tools, data analysis, LaTeX skills, maybe some symbolic computations in Maple, and about careers using these skills. These student meetings can include talks by faculty or other guests speakers with similar interests.
